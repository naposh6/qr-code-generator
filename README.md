# 🚀 GenerQR — Advanced Dynamic QR Code Generator

[![PHP Version](https://img.shields.io/badge/php-%5E8.1-blue.svg)](https://www.php.net/)
[![License](https://img.shields.io/badge/license-MIT-green.svg)](LICENSE)

**GenerQR** — це не просто генератор, а потужний інструмент для створення кастомних QR-кодів. Проєкт реалізовано як фінальну роботу, що демонструє навички написання чистого коду, застосування патернів проєктування та низькорівневої обробки графіки.

---

## 🌟 Ключові особливості (Update v2.0)

Окрім базового функціоналу, у цій версії реалізовано складні технічні рішення:

* **🎨 Custom Dot Styling:** Власний алгоритм перетворення стандартних квадратів QR-коду в ідеальні круглі точки (Dots) за допомогою бібліотеки **GD**.
* **🖼️ Smart Logo Overlay:** Автоматичне накладання логотипу в центр коду з генерацією білої підкладки для 100% читабельності.
* **🔄 Dynamic Content:** Підтримка URL, тексту та автоматичне розміщення медіафайлів (фото/відео) на сервері.
* **🛡️ PHP 8.1+ Optimized:** Повна сумісність з сучасними версіями PHP (виправлено помилки приведення типів та Deprecated notices).
* **⚡ Glassmorphism UI:** Сучасний інтерфейс у стилі Apple з використанням ефектів матового скла та адаптивною версткою.

---

## 🏗️ Архітектура та Патерни (Design Patterns)

Проєкт побудований на принципах **SOLID** та використовує наступні патерни:

* **Singleton:** Єдине підключення до бази даних у `src/Core/Database.php`.
* **Factory Method:** Динамічне створення об'єктів контенту в `src/Factories/QrContentFactory.php`.
* **Repository:** Повна ізоляція логіки БД від контролерів у `src/Repositories/QrRepository.php`.
* **Strategy (Options-based):** Гнучка зміна стилю рендерингу (квадрати/круги) без зміни основного коду сервісу.

---

## 🛠️ Технічний стек

* **Backend:** PHP 8.1+ (Pure PHP без важких фреймворків)
* **Graphic:** GD Library для попіксельної маніпуляції зображеннями
* **QR Engine:** Endroid QR Code (v5.x)
* **Frontend:** HTML5, CSS3 (Modular CSS), JavaScript (Vanilla JS)
* **Database:** MySQL

## 💎 Принципи розробки та Рефакторинг

У проєкті застосовано професійні техніки покращення коду:
- **Extract Method:** Складні алгоритми малювання кружечків розбиті на дрібні приватні методи.
- **Dependency Injection:** Впровадження залежностей через інтерфейс `QrContentInterface`.
- **DRY & KISS:** Мінімізація дублювання коду в сервісах завантаження файлів.
- **Type Hinting:** Суворе використання типів для безпеки даних.

## 🚀 Швидкий старт локально

1.  **Клонування:**
    ```bash
    git clone [https://github.com/naposh6/qr-code-generator.git](https://github.com/naposh6/qr-code-generator.git)
    ```
2.  **Встановлення залежностей:**
    ```bash
    composer install
    ```
    *Примітка: Переконайтеся, що розширення `extension=gd` увімкнено у вашому `php.ini`.*
3.  **Налаштування БД:**
    Створіть базу MySQL та вкажіть дані у `src/Core/Database.php`.
4.  **Права доступу:**
    Дайте права на запис для папки: `public/uploads/qr/`.
